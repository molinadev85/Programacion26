# Sistema de Gestión e Informes de Inventario

Este proyecto es una aplicación web interactiva diseñada para la gestión, análisis e informe en tiempo real de productos e inventarios. Permite llevar un control detallado de existencias, calcular valores monetarios totales y generar alertas de reabastecimiento automáticas.

## 🚀 Características Principales

* *Control de Inventario:* Permite agregar y eliminar productos del listado dinámicamente.
* *Métricas y Análisis:* Muestra el total de productos en stock, el valor financiero monetizado del inventario y alertas visuales para productos con *Stock Bajo (< 5 unidades)*.
* *Generación de Informes en .txt:* Permite exportar y descargar un informe de auditoría detallado con la fecha de emisión, resumen ejecutivo y alerta de reabastecimiento crítico.
* *Interfaz Adaptativa:* Diseño moderno y limpio utilizando CSS3 nativo.
